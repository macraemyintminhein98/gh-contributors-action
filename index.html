GitHub Contributors Action

Automate your README's 'Contributors' section and give credit where it's due.

Get Started on GitHub Marketplace

Effortless Acknowledgment

Automatically lists all unique contributors to your repository.

Always Up-to-Date

Runs on every push, ensuring your README is always current.

Visually Engaging

Displays avatars and links to GitHub profiles for a personal touch.

Open-Source Friendly

Perfect for open-source projects, teams, and developer portfolios.

How It Works

  1. Add the action to your GitHub Workflow (.github/workflows/main.yml).
  2. The action fetches contributors via the GitHub API.
  3. It then updates a specific section in your README.md.
  4. Commit and push – your contributors section is now automated!
